# Kyberlahti 2026: The Translation Layer

This repository contains the source code, presentation scripts, and AI agentic skills used to create the Kyberlahti 2026 presentation.

This presentation was built using a unique hybrid approach, blending human narrative architecture with multi-agent AI collaboration and synthesis.

## Folder Structure

* **`presentation_script.md`**: The final approved speaker script, complete with click cues and pacing notes. It implements the "Lossless Compression" (Anchor, Mechanism, Impact) framework for educating non-technical audiences on complex cybersecurity topics.
* **`react-presentation/`**: The technical implementation. A custom-built React and Framer Motion frontend to deliver high-end, cinematic web-based slide experiences.
* **`agent_skills/`**: The core AI "personas" used to engineer this presentation. These folders demonstrate the system prompts, capabilities, and workflows used by the AI agents that collaborated on this project.

## The Agentic "Translation Layer"
To bridge the gap between deep technical truth and engaging public speaking, we utilized a "Translation Layer" consisting of multiple specialized AI agents:

- **Presentation Expert (Nancy Duarte)**: Sculpted the emotional journey and Sparkline narrative (What Is vs. What Could Be).
- **Core Visuals (Graphics Designer & Artist)**: Engineered the visual communication, cinematic aesthetics, and overall layout.
- **Cinematic Photography (Buffalo 7 & Stinson Design)**: Generated the JSON-structured, hyper-realistic image prompts used for the background visuals.
- **Rhetorician & Social Engineer**: Influenced the pacing, emotional hooks, metaphor selection, and empathy tactics.
- **Developer**: Implemented the bespoke React and Framer Motion frontend for the presentation.
- **Director**: Unified the vision into a cohesive final output to prevent cognitive overload.

## How to use this repository
1. Read the `presentation_script.md` to understand the narrative flow and click cues.
2. Explore `agent_skills/` to see how the AI system was prompted to generate high-end corporate styling and cinematic assets.
3. Run the `react-presentation/` app locally (`npm run dev`) to see how hacker aesthetics and Framer Motion were used to build a bespoke slide deck.
